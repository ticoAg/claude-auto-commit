# 0.1.6 发布到 npm（作用域包）

本次变更将包名从未带作用域的 `claude-auto-commit` 迁移为 npm 作用域包，并以公开方式发布：

- 新包名：`@ticoag/claude-auto-commit`（作用域与包名均为小写，符合 npm 规范）
- `package.json` 增加：
  - `name: "@ticoag/claude-auto-commit"`
  - `publishConfig.access: "public"`

## 安装与使用

```bash
# 一次性执行
npx @ticoag/claude-auto-commit --help

# 全局安装
npm i -g @ticoag/claude-auto-commit

# 使用
claude-auto-commit -l zh -e -c
```

## 注意事项

- 需要 Node.js >= 22
- 建议先完成 `claude login` 以复用已登录的 Claude Code 环境
- 首次在作用域下发布需确保使用 `--access public`（已在 `publishConfig` 中固化）

- 文档：同步更新 README 中的安装命令为作用域包写法
- 代码：未更改功能，仅调整包元数据
