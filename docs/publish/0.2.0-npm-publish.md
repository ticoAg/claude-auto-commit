# 0.2.0 发布到 npm（作用域包）

- 包名：@ticoag/claude-auto-commit
- Node.js：>= 22
- 发布类型：minor

## 主要变更
- 新增：结构化提示词（主题行 + 要点）
- 改进：verbose 分段日志（含 trace_id）
- 改进：提交信息以等号分隔块高亮显示，去除外层引号
- 兼容性：无破坏性改动

## 安装与升级

一次性运行：

```bash
npx @ticoag/claude-auto-commit@0.2.0 --help
```

全局安装：

```bash
npm i -g @ticoag/claude-auto-commit@0.2.0
```

## 发布命令（参考）

已设置 publishConfig.access 为 public，可直接：

```bash
npm publish
```

## 注意事项
- 请确认已登录 npm：
  - `npm whoami`
- 若启用 2FA，按提示输入一次性验证码
- 建议先用 `npm pack --dry-run` 预览打包内容
